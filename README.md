## 🏠 About the project

Este projeto foi desenvolvido como uma landing page visual inspirada na marca **Starbucks**, com foco em apresentação de produto, interatividade simples e construção de uma interface moderna e responsiva.

A proposta da página é destacar o copo principal em tela, junto de uma mensagem de branding, enquanto o usuário pode interagir com miniaturas de produtos para alterar a imagem central e a cor de fundo decorativa da interface.

Durante o desenvolvimento, foram trabalhados conceitos importantes de estruturação com HTML, responsividade com CSS, organização visual, efeitos de hover e manipulação do DOM com JavaScript para troca dinâmica de elementos visuais.

## 🧰 Features

- Hero section com destaque para produto
- Identidade visual inspirada em marca conhecida

## 🎨 Layout and design decisions

O projeto utiliza uma estética limpa e marcante, com muito espaço em branco, destaque para o produto principal e uso de uma forma circular de fundo para reforçar a composição visual.

Alguns pontos de destaque no layout:

- visual minimalista com foco no produto
- tipografia forte para headline principal
- destaque cromático na palavra-chave da marca
- composição equilibrada entre texto e imagem
- uso de miniaturas como elemento de interação
- círculo de fundo para dar profundidade e identidade visual
- adaptação para telas menores sem perder legibilidade

A escolha de manter a interface simples foi acertada, porque o valor do projeto está justamente na clareza visual e na interação direta.

---

## ⚙️ Functional rules

A lógica principal da aplicação funciona assim:

- o usuário visualiza um produto principal em destaque
- ao clicar em uma miniatura:
  - a imagem principal é alterada
  - a cor do círculo de fundo é atualizada
- cada miniatura representa uma variação visual diferente do produto
- a interface responde imediatamente à interação do usuário

---

## 🥤 Product variations

O projeto apresenta três variações visuais de copos, cada uma associada a:

- uma imagem diferente
- uma cor de fundo diferente

Essa troca ajuda a deixar a landing page mais viva e interativa, mesmo com uma lógica simples.
